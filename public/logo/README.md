# Логотип сервиса

Поместите итоговые версии логотипа сюда, чтобы они были доступны в сборке Vite как статические файлы.

- Предпочтительный формат: SVG (`logo.svg`).
- Допустимые варианты для альтернатив: PNG с прозрачным фоном.
- Размеры рекомендуется держать в диапазоне до 512x512 px.

Файлы из этой директории можно подключать через `/logo/<имя-файла>`.
